# Winner Venera — статический сайт

Открой папку в VS Code и запусти любым локальным сервером.

## Быстрый старт (самое простое)
1) Установи расширение **Live Server** в VS Code
2) Открой `index.html` → `Open with Live Server`

## Структура
- `index.html` — главная
- `catalog.html` — каталог (вкладки: гадание/костюмы/атрибуты/ткани)
- `cases.html` — кейсы
- `price.html` — прайс (заказать расчёт)
- `contacts.html` — контакты
- `assets/css/styles.css` — стили
- `assets/js/main.js` — JS (мобильное меню + вкладки)

## Куда класть изображения
Папка: `assets/img/`

Файлы-заглушки, которые можно заменить своими:
- `hero-bg.jpg` — фон первого экрана
- `form-bg.jpg` — фон под формой расчёта
- `about-photo.jpg` — фото слева в блоке «Давайте знакомиться»

Если файлов нет — всё равно будет красиво за счёт градиентов.
